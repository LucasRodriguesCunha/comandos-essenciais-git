GIT INIT

Cria um novo repositório.

GIT CLONE

Clona uma repositório Git existente para um novo diretório.

GIT ADD

Adiciona um arquivo ao índice Git para ser incluído no proxímo commit.

GIT COMMIT

Registra um conjunto de mudanças no repositório Git, criando um novo commit com uma mensagem descritiva.

GIT STATUS

Exibe o estado atual do repositório Git.

GIT PUSH

Envia as mudanças locais para um repositório remoto.

GIT PULL

Obtém as mudanças do repositório remoto e as mescla com as alterações locais.

GIT BRANCH

Cria, lista ou exclui ramos no repositório Git.

GIT CHECKOUT

Altera o ramo atual ou o arquivo de trabalho para um commit específico.

GIT MERGE

Combina as mudanças de dois ou mais ramos em um único ramo.

GIT LOG

Exibe um histórico de commits no repositório Git.

GIT STASH

Salva temporariamente mudanças em um diretório de trabalho sem fazer commit.

GIT DIFF

Exibe as diferenças entre duas versões de um arquivo ou entre dois ramos.

GIT TAG

Marca um commit específico como importante.

GIT RESET

Desfaz mudanças em um arquivo ou conjunto de arquivos para um commit específico.

GIT RM

Após executar o comando, o arquivo será excluído do diretório de trabalho e do controle de versão do Git.
